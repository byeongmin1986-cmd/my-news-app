# KODEX SK하이닉스단일종목레버리지 ETF 대시보드

전일 종가 대비 매수·매도 기준가를 자동 계산하고, 3천만원 7분할 매수/매도 계획을 시각화하는 Streamlit 대시보드입니다.

## 주요 기능

- **전일 종가 자동 조회** (네이버 금융 → Daum 금융 → pykrx 순서로 시도, 실패 시 직접 입력)
- **매수 기준가**: 전일 종가 × (1 + 매수 상승률)
- **매도 목표가**: **실제 매수가 기준** × (1 + 매도 상승률) — 수익 구조 보장
- **5원 단위 호가** 처리 (올림/반올림 선택)
- **N분할 매수/매도 계획표** (기본: 3천만원 7분할)
- 현재가 입력 시 매수·매도 기준 도달 여부 표시

## 매도 기준가 로직

> 전일 종가 대비 +3.6%에 매수하고 +3.5%에 매도하면 손실입니다.  
> 수익 구조를 위해 **매도 기준은 실제 매수가 기준**으로 계산합니다.

```
buy_price  = ceil_to_5(prev_close × 1.036)   # 전일 종가 +3.6%
sell_price = ceil_to_5(buy_price  × 1.035)   # 매수가 +3.5% → 수익
```
